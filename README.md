# Godot Dictionary

This repository acts as a reference and hub.
I intend to fill it with implementations of useful features within Godot.


### Grid-Based Movement
#### Overview
* Creates a grid object representing a possible tilemap
* Optional grid display script
* Click-Based movement while aligned on grid

#### Camera Movement Schemes
1. Centered around player node
2. controlled via WASD, changeable speed/distance variables
3. Free movement withing bounds, movement of player past offset shifts camera
scheme 3 seems most appropriate
